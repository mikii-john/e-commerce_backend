### üõ†Ô∏è API Integration (Crucial):
The frontend must integrate with the following backend endpoints (Base URL: http://localhost:5000):

1. **Get All Products**: `GET /api/products`
   - Returns: `{ success: true, data: [ { id, name, description, price, category, image_url, stock } ] }`
   - Role: Used for the main shop landing page and product grids.

2. **Get Product by ID**: `GET /api/products/:id`
   - Returns: `{ success: true, data: { product_details } }`
   - Role: Used for the Product Detail page.

3. **Get Products by Category**: `GET /api/products/category/:category`
   - Returns: `{ success: true, data: [ filtered_products ] }`
   - Role: Used for category filtering and navigation.

4. **Place Order**: `POST /api/orders`
   - Body: `{ customer_email: string, items: [ { product_id: string, quantity: number } ] }`
   - Returns: `{ success: true, message: "Order placed successfully!", data: order_details }`
   - Role: Used at checkout to submit the cart.

5. **Get All Orders**: `GET /api/orders` (Optional/Admin)
   - Role: Track existing orders.

6. **Health Check**: `GET /api/health`
   - Role: Display system status indicators in an admin dashboard or footer.
